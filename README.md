# 🏨 New Express Restaurant API

A **RESTful backend server** for a restaurant application built using **Node.js**, **Express.js**, and **MongoDB (Mongoose)**.  
Designed for managing restaurants and providing API endpoints to interact with data via Postman or frontend apps.

---

## 📌 Features

✔️ Express.js server setup  
✔️ MongoDB database integration using Mongoose  
✔️ MVC project structure (controllers, routes, models)  
✔️ REST API endpoints  
✔️ Easy testing with Postman or any API client  
✔️ Reactive and smooth frontend
